## 🐛 问题描述 | Problem Description

### 问题一：注册后自动登录 | Auto-login After Registration

**现象 | Symptoms:**
- 患者完成注册后，系统自动登录并跳转到患者端主页
- 期望行为：注册成功后返回登录页面，由用户手动登录

**根本原因 | Root Cause:**
- 后端注册API返回登录token（auth.py）
- 前端AuthContext存储token并自动跳转（AuthContext.tsx）

### 问题二：地址信息未保存 | Address Not Saved

**现象 | Symptoms:**
- 注册页面填写的地址信息在注册后丢失
- 进入"个人中心"需要重新填写地址
- 用户表的address字段为空

**根本原因 | Root Cause:**
- UserCreate schema缺少address字段（user.py）
- 注册端点未将address传递给UserService（auth.py）
- UserService.create_user方法不接受address参数

---

## 🔧 修复方案 | Fix Solution

### 修复一：移除自动登录 | Remove Auto-login

**后端修改 | Backend Changes:**

1. **auth.py** - 注册端点不再返回token
```python
# 修改前 | Before:
return {"user": user, "tokens": tokens}

# 修改后 | After:
return {
    "message": "注册成功，请登录",
    "user_id": str(user.id),
    "email": user.email,
}
```

**前端修改 | Frontend Changes:**

2. **AuthContext.tsx** - 注册后不存储token
```typescript
// 修改前 | Before:
localStorage.setItem(CONFIG.TOKEN_KEY, response.tokens.access_token);
localStorage.setItem(CONFIG.REFRESH_TOKEN_KEY, response.tokens.refresh_token);
setUser(response.user);
window.location.href = roleRoutes[response.user.role] || '/';

// 修改后 | After:
// 注册成功，不自动登录，不存储token
// 由调用方（RegisterPage）处理跳转
```

### 修复二：保存地址信息 | Save Address

**后端修改 | Backend Changes:**

1. **user.py** - UserCreate添加address字段
```python
class UserCreate(UserBase):
    password: str
    date_of_birth: Optional[str] = None
    gender: Optional[str] = None
    phone: Optional[str] = None
    address: Optional[str] = None  # ✅ 新增
    emergency_contact_name: Optional[str] = None
    emergency_contact_phone: Optional[str] = None
```

2. **auth.py** - 注册时传递address
```python
# 创建用户时传递地址
user = await user_service.create_user(
    email=user_data.email,
    password=user_data.password,
    full_name=user_data.full_name,
    address=user_data.address,  # ✅ 新增
)

# 创建患者档案时传递地址
patient_data = PatientCreate(
    date_of_birth=date_of_birth,
    gender=user_data.gender,
    phone=user_data.phone,
    address=user_data.address,  # ✅ 新增
    emergency_contact=emergency_contact,
)
```

3. **user_service.py** - create_user接受address参数
```python
async def create_user(self, email: str, password: str, full_name: str, address: Optional[str] = None) -> User:
    # ...
    user = User(email=email, password_hash=password_hash, full_name=full_name, address=address)
```
